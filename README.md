# poke_counter
Pokemon information searcher. It finds counter Pokemon as well.

# Goals
- [x] ID
- [x] Type
- [x] Stats
- [x] Abilities
- [x] Evolution Level
- [ ] Evolution Method (Friendship, Item)
- [ ] Move table (HM/TM/TR/Egg/Move Tutor)
- [ ] Counter Type(attack/defense)
- [ ] Counter Pokemon
